# 🏍️ MotoRent: Scooter & Bike Rental

A two-page web application for tourists to browse and rent scooters & motorcycles at vacation spots.

> **Case Study 36** | ITM Skills University | B.Tech CSE 2025–29 | Semester II

---

## 📌 About

MotoRent is a rental website built with HTML5, CSS3, and JavaScript. Tourists can browse a vehicle catalog, compare options, check pricing, and verify their driving license — all from a single responsive web app.

---

## ✨ Features

| Page | Features |
|------|----------|
| **vehicles.html** | Vehicle catalog, dynamic JS filtering, CSS comparison table, pricing plans, included amenities |
| **verification.html** | License verification form, regex validation, age check, real-time feedback, booking ID |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure, forms, tables
- **CSS3** — Grid, Flexbox, variables, comparison table, dark theme, animations
- **JavaScript (ES6)** — DOM manipulation, regex validation, filtering, event listeners
- **Google Fonts** — Outfit + Space Mono

No frameworks or build tools required.

---

## 🚀 How to Run

1. Clone the repo
2. Open the folder in VS Code
3. Right-click `index.html` → **Open with Live Server**

Or just open `index.html` directly in any browser.
